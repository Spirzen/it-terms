## Backend

Серверная часть программного обеспечения, отвечающая за реализацию бизнес-логики, обработку данных, взаимодействие с базами данных, внешними API и другими сервисами. Обычно скрыта от конечного пользователя и взаимодействует с frontend через сетевые протоколы (например, HTTP/REST, WebSocket, gRPC).

---

## back-end database

back-end database — база данных заднего плана, центральное понятие двухзвенных приложений, в которых действие разделяется на front-end (передний план), отвечающий за взаимодействие с пользователем и необходимые расчеты и back-end, где хранятся данные. По сравнению с более ранним подходом к разработке приложений, в котором программы использовали встроенную базу данных и вся программная часть SQL-запросов хранилась на сервере баз данных в виде хранимых процедур или триггеров.

## Bash

Bourne-Again SHell — командная оболочка и язык сценариев UNIX-подобных операционных систем. Является расширенной заменой оригинальной Bourne Shell (sh). Широко используется для автоматизации задач, системного администрирования и написания скриптов в Linux и macOS.

Напоминалка по конструкциям `()`, `{}`, `$()`, `[[ ]]` — [Знаки препинания в командной строке](/encyclopedia/2-system-network/2-05-terminal/11#napominalka-bash), раздел [Bash](/encyclopedia/5-languages/5-25-bash/intro).

## Bash script

Сценарий на языке командной оболочки Bash. Представляет собой текстовый файл с последовательностью команд, исполняемых интерпретатором Bash. Используется для автоматизации системных задач в UNIX-подобных ОС. Обычно имеет расширение `.sh` и исполняемый бит.

## BCL

BCL — Base Class Library, базовая библиотека классов в .NET. Это ядро стандартной библиотеки, включающее фундаментальные типы и сервисы, необходимые для любого приложения: System.Object, String, Array, коллекции (`List<T>`, `Dictionary<TKey, TValue>`), исключения, потоки (Stream, FileStream), примитивы синхронизации (Monitor, Mutex), атрибуты, отражение (Type, Assembly), сериализация, даты и время, числа. BCL — часть Common Language Infrastructure (CLI) и реализуется единообразно во всех .NET-платформах: .NET Framework, .NET Core, Mono, Xamarin. Важно не путать с FCL (Framework Class Library) — расширенной библиотекой, включающей BCL плюс высокоуровневые API (Windows Forms, WPF, ASP.NET). В .NET 5+ термин FCL упразднён; говорят просто о standard libraries, где BCL — её неизменная основа.

## BCLK

Base Clock (базовая тактовая частота) — опорная частота, от которой производятся все остальные тактовые сигналы в системе: CPU, RAM, PCIe, USB.

## Beta

Beta-версия обычно стабильна, содержит почти весь функционал и распространяется среди широкой аудитории (публичная бета) или ограниченного круга тестировщиков (закрытая бета) для выявления ошибок и сбора обратной связи.

## Big Data

Наборы данных, объём, скорость поступления или разнообразие которых превышают возможности традиционных систем управления данными.

## BIOS

Basic Input/Output System — встроенная микропрограмма на материнской плате компьютера, обеспечивающая начальную инициализацию аппаратных компонентов при включении и загрузку операционной системы. Современные системы постепенно заменяют BIOS на UEFI.

## Bioware

BioWare — канадская студия-разработчик ролевых видеоигр, принадлежащая Electronic Arts. Известна такими франшизами, как *Mass Effect*, *Dragon Age*, *Star Wars: Knights of the Old Republic*. Характеризуется сильным нарративом, системой диалогов и морального выбора, а также партийной механикой.

## Bit

Binary digit — наименьшая единица информации в цифровых системах. Принимает одно из двух значений: 0 или 1. Является основой двоичной системы счисления, используемой в вычислительной технике.

---

## Bitmap

Растровое изображение, представленное в виде массива битов, где каждый пиксель кодируется определённым количеством бит (глубина цвета). Форматы, основанные на bitmap: BMP, PNG, JPEG. Противопоставляется векторной графике.

## Blender

Blender — профессиональное свободное и открытое программное обеспечение для создания трёхмерной компьютерной графики, включающее в себя средства моделирования, скульптинга, анимации, симуляции, рендеринга, постобработки и монтажа видео со звуком, компоновки с помощью "узлов" (Node Compositing), а также создания 2D-анимаций. В настоящее время пользуется большой популярностью среди бесплатных 3D-редакторов в связи с его быстрым стабильным развитием и технической поддержкой.

Известна созданием культовых франшиз: *Warcraft*, *StarCraft*, *Diablo*, *World of Warcraft*, *Overwatch*. Отличается высоким качеством полировки, баланса и долгосрочной поддержки проектов.

## Blockchain

Цифровая распределённая база данных (реестр), в которой транзакции объединяются в блоки и связываются в цепочку с использованием криптографических хешей. Обеспечивает неизменяемость, прозрачность и децентрализованную проверку без доверенного посредника. Основа криптовалют и многих децентрализованных приложений.

## Batch processing

Batch processing (пакетная обработка) — выполнение задач **группами** (батчами), а не по одной записи или в реальном времени. В интеграциях: ночной ETL, массовый `POST /resource/batch`, JDBC batch insert, батчинг продюсера Kafka (`batch.size`, `linger.ms`). Снижает накладные расходы сети и СУБД, но увеличивает задержку и требует watermark, checkpoint и идемпотентной загрузки. Хаб: [Пакетная работа с данными](/encyclopedia/3-data-markup/3-11-analiz-dannyh/433); также [пакетная загрузка в интеграциях](/encyclopedia/2-system-network/2-09-osnovy-integratsionnogo-vzaimodeystviya/112#batch-etl-load), [batch в REST API](/encyclopedia/2-system-network/2-09-osnovy-integratsionnogo-vzaimodeystviya/117#rest-api-design-principles), [batch vs stream](/encyclopedia/3-data-markup/3-11-analiz-dannyh/423).

## Bulk operation

Bulk operation (массовая операция) — запись, обновление или удаление **многих записей одним вызовом** API или СУБД (`bulk insert`, `COPY`, `insertMany`, `_bulk` в Elasticsearch). Уровень оптимизации I/O, внутри которого часто используют **чанки**. См. [Пакетная работа с данными](/encyclopedia/3-data-markup/3-11-analiz-dannyh/433#sql-bulk).

## BaaS

Backend as a Service — облачная модель предоставления серверной инфраструктуры для мобильных и веб-приложений. Поставщики BaaS (например, Firebase, AWS Amplify) предлагают готовые решения для аутентификации, хранения данных, уведомлений и аналитики без необходимости развёртывания собственного backend.

---

## BCI

Brain-Computer Interface — интерфейс "мозг-компьютер", система, обеспечивающая прямую связь между нейронной активностью мозга и внешним устройством. Использует ЭЭГ, имплантируемые сенсоры или другие методы регистрации сигналов. Применяется в медицине, исследованиях и экспериментальных HCI-решениях.

---

## Bandwidth

Полоса пропускания: максимальный объём данных, который может быть передан по каналу связи за единицу времени. Измеряется в битах в секунду (бит/с, Мбит/с и т.д.). Часто ошибочно используется как синоним "скорости интернета", хотя технически относится к пропускной способности канала.

---

## Binary

Двоичная система счисления, используемая в цифровых устройствах, где вся информация представлена последовательностями из нулей и единиц. Также может обозначать исполнимый файл (binary executable) в отличие от исходного кода.

---

## Blink

Blink — движок отрисовки (rendering engine), форк WebKit, используемый в Chromium и производных.

---

## Bluetooth

Стандарт беспроводной связи малого радиуса действия (до 10–100 м), предназначенный для обмена данными между периферийными устройствами (наушники, клавиатуры, смартфоны). Работает в диапазоне 2.4 ГГц, использует частотное расширение спектра. Управляется Bluetooth SIG.

---

## BOM (Browser Object Model)

BOM — Browser Object Model, модель объектов браузера. Это неформальный термин, обозначающий совокупность глобальных JavaScript-объектов и API, предоставляемых средой выполнения веб-браузера помимо DOM (Document Object Model). DOM описывает структуру документа; BOM — взаимодействие с окружением: окном, историей, локацией, таймерами, хранилищами. 

---

## Boolean

Boolean — логический тип данных: true / false.

---

## Boot

Процесс инициализации компьютера после включения питания, включающий выполнение микропрограммы (BIOS/UEFI), обнаружение оборудования, загрузку загрузчика (например, GRUB) и последующую загрузку операционной системы в оперативную память.

---

## Boot Camp

Утилита от Apple, позволяющая устанавливать и запускать операционную систему Microsoft Windows на компьютерах Mac с архитектурой Intel. Не поддерживается на устройствах с чипами Apple Silicon (M1 и новее). Предоставляет драйверы и инструменты для совместимости оборудования.

---

## Bootstrap

Bootstrap (также известен как Twitter Bootstra) — популярный фреймворк для создания сайтов и веб-приложений с открытым исходным кодом. Включает в себя HTML- и CSS-шаблоны оформления для типографики, веб-форм, кнопок, меток, блоков навигации и прочих компонентов веб-интерфейса, включая JavaScript-расширения.

---

## Boolean

Тип данных в программировании и логике, принимающий одно из двух значений: *true* (истина) или *false* (ложь). Назван в честь Джорджа Буля, основателя булевой алгебры. Используется в условных выражениях, циклах и логических операциях.

---

## Borderlands

Borderlands: Серия шутеров от Gearbox Software, сочетающая элементы RPG и "looter-shooter" — механики сбора большого количества генерируемых случайным образом предметов (в первую очередь оружия). Известна стилизованной под комикс визуальной подачей и кооперативным геймплеем.


---

## Bot

Сокращение от *robot* — программный агент, выполняющий автоматизированные задачи. Может использоваться легально (поисковые роботы, чат-боты) или в злонамеренных целях (спам-боты, боты в играх). В кибербезопасности — компонент ботнета.


---

## BPMN

BPMN (англ. Business Process Model and Notation, нотация и модель бизнес-процессов) — система условных обозначений (нотация) и их описания в XML для моделирования бизнес-процессов. Разработана Business Process Management Initiative (BPMN.org) и поддерживается Object Management Group, после слияния обеих организаций в 2005 году. Последняя версия BPMN — 2.0 (2.0.2), предыдущая версия — 1.2.

---

## Brb

Сокращение от англ. *Be right back* — "сейчас вернусь". Используется в онлайн-общении для обозначения кратковременного отсутствия. Часто встречается в чатах, голосовых звонках и многопользовательских играх.

---

## Branch (Git)

В системе контроля версий Git — независимая линия разработки, позволяющая изолировать изменения. Основная ветка обычно называется `main` или `master`. Ветвление используется для разработки новых функций, исправления ошибок или экспериментов без нарушения стабильности основного кода.

---

## Branch Unit

Блок процессора, реализующий механизмы предсказания ветвлений (branch prediction) и управления потоком исполнения. 

---

## BSON

Двоичный формат представления данных, расширяющий JSON возможностью хранить дополнительные типы, включая даты и двоичные объекты.

---

## Btrfs

B-tree File System — современная файловая система для Linux, разработанная Oracle. Поддерживает копирование при записи (COW), снапшоты, сжатие, встроенный RAID, проверку целостности и онлайн-дефрагментацию. Предназначена для масштабируемости и надёжности при работе с большими объёмами данных.

---

## Buffer

Область памяти, используемая для временного хранения данных при передаче между процессами, устройствами или компонентами системы с различной скоростью обработки. Применяется в потоковом видео, сетевой передаче, вводе-выводе и графических API.

---

## Bug

Ошибка, недочёт или несоответствие в программном обеспечении, приводящее к некорректному поведению, сбоям или уязвимостям. Термин восходит к эпизоду с молью в реле компьютера Harvard Mark II в 1947 году.

---

## Build

Собранный экземпляр программного обеспечения, полученный в результате компиляции исходного кода и связывания зависимостей. Может относиться к версии для тестирования, релиза или развёртывания. В играх — также может обозначать конкретную сборку персонажа (набор навыков и экипировки).

---

## Byte

Единица хранения данных, состоящая из 8 бит (в современных архитектурах). Может представлять один символ в кодировках ASCII или UTF-8, целое число в диапазоне 0–255 (беззнаковое) или −128…127 (знаковое).
