## N-API

N-API — стабильный API для создания нативных аддонов к Node.js на C/C++. Обеспечивает бинарную совместимость между версиями Node.js без перекомпиляции, в отличие от устаревшего Nan.

---

## NaN

NaN — Not a Number, специальное значение с плавающей точкой, определённое в стандарте IEEE 754 для обозначения неопределённых или некорректных результатов вычислений.

---

## NAT

NAT (Network Address Translation) — метод трансляции IP-адресов в сетях, позволяющий нескольким устройствам в локальной сети использовать один публичный IP-адрес для доступа в интернет. Реализуется на маршрутизаторах и межсетевых экранах. Типы: SNAT, DNAT, PAT.

---

## Need For Speed

Need for Speed — серия аркадных гоночных игр от Electronic Arts, посвящённая уличным гонкам, тюнингу автомобилей и погоням с полицией. Одна из старейших и самых продолжительных игровых франшиз (с 1994 года).

---

## Nerd

Nerd — в культурном контексте IT: человек, глубоко увлечённый техникой, наукой, программированием или поп-культурой, часто ассоциируется с высокой компетентностью в узкой области и скромным социальным поведением. В современном употреблении — нейтральный или позитивный термин.

---

## .NET

.NET (ранее известна как .NET Core) — модульная платформа для разработки программного обеспечения с открытым исходным кодом.

---

## .NET Framework

.NET Framework — программная платформа, выпущенная компанией Microsoft в 2002 году. Основой платформы является общеязыковая среда исполнения Common Language Runtime (CLR), которая подходит для различных языков программирования. Функциональные возможности CLR доступны в любых языках программирования, использующих эту среду. В настоящее время .NET Framework развивается в виде .NET.

---

## NetBeans IDE

NetBeans IDE — свободная интегрированная среда разработки приложений (IDE) на языках программирования Java, Python, PHP, JavaScript, C, C++, Ада и ряда других.

---

## Neverwinter Nights

Neverwinter Nights — серия RPG от BioWare и Obsidian Entertainment, основанная на правилах Dungeons & Dragons 3.5. Известна мощным редактором модов (Aurora Toolset), позволявшим создавать пользовательские кампании и мультиплеерные серверы.

---

## NFS

NFS (Network File System) — распределённая файловая система, разработанная Sun Microsystems, позволяющая монтировать удалённые каталоги по сети как локальные. Широко используется в Unix-подобных системах для совместного доступа к данным.

---

## NFC

NFC (Near Field Communication) — стандарт беспроводной связи малого радиуса (до 10 см), работающий на частоте 13.56 МГц. Применяется для бесконтактных платежей (Apple Pay, Google Pay), обмена данными и взаимодействия с RFID-метками.

---

## Nginx

Nginx (engine x — произносится как энджи́нкс или э́нжин-и́кс) — веб-сервер и почтовый прокси-сервер, работающий на Unix-подобных операционных системах (тестировалась сборка и работа на FreeBSD, OpenBSD, Linux, Solaris, macOS, AIX и HP-UX). С версии 0.7.52 существует экспериментальная сборка под Microsoft Windows.

---

## NIC

NIC (Network Interface Controller / Card) — аппаратный компонент (встроенный или сменный), обеспечивающий подключение компьютера к компьютерной сети. Имеет уникальный MAC-адрес и поддерживает протоколы канального уровня (Ethernet, Wi-Fi).

---

## Nintendo

Nintendo — японская компания-разработчик видеоигр и игровой техники, основанная в 1889 году. Известна созданием культовых франшиз (Super Mario, The Legend of Zelda, Pokémon, Animal Crossing) и инновационными консолями (NES, Game Boy, Wii, Switch).

---

## NIO.2

NIO.2 — New I/O 2, второе поколение API ввода-вывода в Java (введено в Java 7 как часть JSR 203). Замена устаревшего java.io и частично — оригинального NIO (java.nio).

---

## NLP

NLP (Natural Language Processing — обработка естественного языка) — раздел искусственного интеллекта, посвящённый взаимодействию компьютеров и человеческого языка. 

---

## Node.js

Node.js — кроссплатформенная среда выполнения JavaScript вне браузера, построенная на движке V8 от Google. Использует событийно-ориентированную архитектуру и неблокирующий I/O, что делает её эффективной для масштабируемых серверных приложений и микросервисов.

---

## NoSQL

NoSQL (Not only SQL) — класс систем управления базами данных, отказывающихся от реляционной модели в пользу гибких схем и горизонтального масштабирования. Типы: документные (MongoDB), ключ-значение (Redis), колоночные (Cassandra), графовые (Neo4j). Подходят для больших объёмов неструктурированных данных.

---

## npm

npm (Node Package Manager) — менеджер пакетов для JavaScript и стандартный реестр для Node.js. Позволяет устанавливать, обновлять и управлять зависимостями проекта. Является крупнейшим в мире репозиторием программного обеспечения с открытым исходным кодом.

---

## NPC

NPC (Non-Player Character) — персонаж в видеоигре, контролируемый искусственным интеллектом или сценариями, а не человеком. Может выполнять функции торговца, союзника, врага или источника сюжетной информации.

---

## NTP

NTP (Network Time Protocol) — сетевой протокол для синхронизации системного времени между компьютерами в сети с высокой точностью (до миллисекунд и ниже). Использует иерархию серверов времени (stratum levels) и работает поверх UDP.

---

## NTT

NTT (Number Theoretic Transform — теоретико-числовое преобразование) — обобщение быстрого преобразования Фурье (FFT) на конечные поля (кольца вычетов по модулю простого числа). В отличие от FFT, оперирующего комплексными числами, NTT работает целочисленно, что исключает ошибки округления и критично для криптографии. 

---

## NuGet

NuGet (произносится как "New Get") (изначальное название "NuPack") — это бесплатный пакетный менеджер с открытым исходным кодом, служащий для .NET и .NET Core механизмом совместного использования кода, поддерживаемым Microsoft. Он определяет, как создаются, размещаются и используются пакеты для .NET, а также предоставляет средства для каждой из этих ролей.

---

## Null

Null — специальное значение в программировании, обозначающее отсутствие ссылки на объект или неинициализированное состояние. В языках со строгой типизацией (C#, Java) отличается от нуля и пустой строки. Может быть источником ошибок (null pointer exception), если не обрабатывается явно.

---

## NumPy

NumPy — фундаментальная библиотека для научных вычислений на языке Python. Предоставляет многомерные массивы (`ndarray`), оптимизированные математические функции и инструменты линейной алгебры. Является основой для экосистемы данных Python (Pandas, SciPy, scikit-learn). Примеры с разбором — [NumPy — массивы и матрицы](/lab/Примеры/1129); теория — [Анализ данных — pandas, NumPy, SciPy](/encyclopedia/5-languages/5-02-python/33).

---

## NUMA

NUMA (Non-Uniform Memory Access — неоднородный доступ к памяти) — архитектура многопроцессорных систем, в которой память физически разделена на банки, "привязанные" к отдельным CPU (или сокетам). Доступ к "локальной" памяти (своего банка) быстрее и имеет меньшую задержку, чем к "удалённой" (банку другого CPU). NUMA возникает естественно при масштабировании SMP-систем. 

---

## NUnit

NUnit — открытая среда юнит-тестирования приложений для .NET. Она была портирована с языка Java (библиотека JUnit). Первые версии NUnit были написаны на J#, но затем весь код был переписан на C# с использованием таких новшеств .NET, как атрибуты.

---

## NVLink

NVLink — высокоскоростной интерконнект, разработанный NVIDIA для соединения GPU между собой и с CPU (в системах на базе POWER или Grace). 

---

## NVMe

NVMe (Non-Volatile Memory Express) — протокол и интерфейс для высокоскоростных SSD, подключаемых через PCIe. Обеспечивает значительно более высокую пропускную способность и низкую задержку по сравнению с SATA и AHCI, оптимизирован для параллельной обработки команд.

---

## NVR

NVR (Network Video Recorder) — система видеонаблюдения, записывающая потоки с IP-камер по сети. В отличие от DVR, обработка видео (кодирование) происходит на самих камерах, а NVR только управляет записью и хранением.
